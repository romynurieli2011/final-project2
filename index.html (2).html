מתכון לפנקייקים

שלבי ההכנה:

  1. שמים בקערה את המצרכים: קמח, סוכר, מלח, אבקת אפייה ותמצית וניל ומערבבים
  2. יוצרים גומה ומוסיפים לתוכה חלב, ביצה, שמן, שמן, תמצית וניל ומערבבים
  3. מערבבים בעזרת מטרפה עד שאין גושים ומחכים חמש דקות
  4. מחממים מחבת עם שמן ויוצקים את הבלילה, מטגנים עד שנוצרות בועות ואז הופכים לצד השני ומטגנים עוד דקה
  5. מגישים את הפנקייקים עם רוטבים שאוהבים ותוספות שרוצים
